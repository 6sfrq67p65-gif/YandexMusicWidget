# Yandex Music Mini Player (macOS) 🎵

![Yandex Music Widget Banner](https://placehold.co/800x400/18181b/ffffff?text=Yandex+Music+Mini+Player+for+macOS&font=Montserrat)

Изящный, легкий и невероятно красивый плавающий виджет Яндекс Музыки для macOS. Разработан на Swift с использованием нативных технологий Apple и WebView.

## ✨ Почему этот виджет лучше аналогов?

Большинство существующих решений для Яндекс Музыки на Mac делятся на две категории:
1. **Официальное десктопное приложение:** Громоздкое, занимает много места на экране и часто избыточно для простой фоновой музыки во время работы.
2. **Браузерные расширения/вкладки:** Постоянно теряются среди десятков открытых вкладок. Приходится искать нужную, чтобы просто поставить трек на паузу.

**Наш мини-плеер решает эти проблемы:**
* 🪟 **Всегда под рукой:** Плавающее окно поверх всех рабочих пространств.
* 🎨 **Нативная эстетика Apple:** Полупрозрачный фон с эффектом матового стекла (Vibrancy), идеальная интеграция в дизайн macOS.
* 🔄 **Два режима отображения:** Вертикальный (классический виджет) и Горизонтальный (максимально компактный для экономии места на экране).
* ⚡️ **Никаких костылей с авторизацией:** Плеер работает на базе изолированного WebView. Вы просто логинитесь в свой Яндекс Аккаунт один раз, и сессия надежно сохраняется.
* 🛡 **Бронебойная синхронизация:** Уникальный 5-уровневый алгоритм сбора метаданных (через скрытый Yandex API, Media Session API и прямого DOM-парсинга). Даже если Яндекс обновит дизайн своего сайта, плеер продолжит показывать время и переключать треки.

## 🚀 Установка (Для пользователей)

Не нужно ничего компилировать или устанавливать Xcode! 

1. Прямо здесь, на главной странице репозитория (в списке файлов выше), найдите файл **`YandexMusicWidget.zip`**.
2. Нажмите на него, а затем на кнопку **Download** (Скачать) справа.
3. Распакуйте скачанный архив — внутри будет готовое приложение.
4. Перетащите иконку приложения в папку **«Программы» (Applications)**.
5. Запустите приложение. macOS может заблокировать первый запуск (так как это приложение от стороннего разработчика) — просто зайдите в **Системные настройки -> Конфиденциальность и безопасность**, прокрутите вниз и нажмите **"Все равно открыть"**.

## 🛠 Сборка из исходников (Для разработчиков)

Если вы хотите внести изменения в код:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ВАШ_НИК/YandexMusicWidget.git
   ```
2. Откройте файл `YandexMusicWidget.xcodeproj` в Xcode.
3. Выберите таргет вашего Mac и нажмите `Cmd + R` для сборки и запуска.
4. **Важно:** При внесении изменений в UI рекомендуем делать Clean Build (`Cmd + Shift + K`), чтобы избежать залипания кэша.

## 🏗 Архитектура и технологии

* **SwiftUI:** Весь интерфейс построен на декларативном фреймворке Apple.
* **WKWebView:** Изолированная среда для рендеринга плеера с кастомным User-Agent.
* **JavaScript Инъекции:** Двусторонняя связь между Swift-моделью и DOM-деревом Яндекса для извлечения метаданных треков в реальном времени.
